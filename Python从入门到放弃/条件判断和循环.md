# 条件判断和循环
### 条件判断和嵌套
```Python
成绩 = int(input ('请输入你的成绩：'))
if 成绩 <= 100 :
    if 成绩 >= 90 :
        print ('A')
    elif 成绩 >= 80 :
        print ('B')
    elif 成绩 >= 70 :
        print ('C')
    elif 成绩 >= 60 :
        print ('D')
    else :
        print ("不及格！！！")
else :
    print ('吹吧你！')
```


### 死循环
```python
while True:
    answer = input('请问我可以退出循环了吗？')
    if answer == '可以':
        break
    print ('哎，好累～～～')
```
