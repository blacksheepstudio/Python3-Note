**在Python程序中，如果我们把单个或多个字符用单引号或者双引号包围起来，就可以表示一个字符串。**
```python
s1 = 'hello, world!'
s2 = "hello, world!"
# 以三个双引号或单引号开头的字符串可以折行
s3 = """
hello, 
world!
"""
print(s1, s2, s3, end='')
```
**为什么设置两种引号？**
- 单引号作用
```python
print('BlackSheep:“Hello world!”')

输出结果 ➡️ BlackSheep:“Hello world!”
```
- 双引号作用
```python
print("Let's Python!")

输出结果 ➡️ Let's Python!
```
