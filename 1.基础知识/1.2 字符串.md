**在Python程序中，如果我们把单个或多个字符用单引号或者双引号包围起来，就可以表示一个字符串。**
```python
s1 = 'hello, world!'
s2 = "hello, world!"
# 以三个双引号或单引号开头的字符串可以折行
s3 = """
hello, 
world!
"""
print(s1, s2, s3)
```
**❓ 为什么Python支持两种引号**
```python
#单引号作用
print('BlackSheep:“Hello world!”')
#➡️ BlackSheep:“Hello world!”

#双引号作用
print("Let's Python!")
#➡️ Let's Python!

#混合输出
print("Let's Python!")
```
